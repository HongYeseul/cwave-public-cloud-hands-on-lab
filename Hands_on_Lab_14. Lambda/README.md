### 1. Lambda 함수 생성

- **Lambda 메인 콘솔 화면 → "함수 생성" 버튼 클릭**

- 함수 생성 정보 입력

    - 함수 이름: lab-edu-lambda-event-handler

    - 런타임: python 3.10

    - '함수 생성' 버튼 클릭

        ![alt text](./img/lambda_01.png)
